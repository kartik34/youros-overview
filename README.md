# youros-overview

**yourOS** is a nostalgic, Windows 95–inspired website generator. It lets people create small desktop-style pages filled with photos, notes, music, and little apps. It's perfect as a gift, a memory box, or a personal homepage.

---

You can:

- Add photos, songs, notes, and apps  
- Pick themes and icons  
- Publish to a unique URL  
- Share it with someone special or use it as your own personal space  

---

## Why I’m building it

I wanted to let people make websites that feel fun to explore, and fun to customize. The modern web should be interactive, and not so static.
